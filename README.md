# Web Representation 🚀

这是一个专为 AI 时代打造的极简、单文件 HTML 幻灯片（PPT）演示框架。无需复杂的构建工具，没有冗杂的 Node.js 依赖，只需一个文件，就能让 AI 编程助手完美接管。

## 🌟 为什么需要这个框架？

传统的 PPT/Keynote 极度依赖手动拖拽排版。而现代网页幻灯片框架（如 Reveal.js, Slidev）虽然强大，但需要配置本地环境和复杂语法，AI 代理（Agent）很容易在多文件结构中迷失方向，输出错误代码。

**Web Representation** 旨在解决这一痛点。它提供了一个基于现代 CSS 容器查询 (`cqw`) 构建的独立 HTML 模板，天生就是 AI Agent 绝佳的“设计画布”。

## ⚡ 使用方式 (Workflow)

使用流程简单到令人发指：

1. **下载文件**：克隆或下载本仓库的 `index.html` 和默认的设计规范 `web-presentation-design.md`。
2. **选择设计风格 (可选)**：你可以去 [getdesign.md](https://getdesign.md/) 找一个你自己喜欢的设计风格（MD 文件），替换掉默认的设计规范。当然，不换也完全没问题，本项目自带了一套极具高级感和商务范的“星巴克绿”设计。
3. **写个提纲**：随便新建一个 Markdown，写下你这次要汇报的提纲和内容要点。
4. **丢给 Agent**：把你下载的文件、设计规范，以及你写的提纲，一起扔给你最喜欢的 AI 编程助手（比如 Cursor, Gemini, Claude 等），然后告诉它：_“请严格参考设计规范文件，并根据我的提纲，在 index.html 里生成我的演示文稿。”_
5. **准备演示图片 (可选)**：如果汇报中需要展示超高清原图，**请将图片直接存放在与 `index.html` 同级的目录下**。单图直接命名为对应页码（如 `6.png`）；同一页有多图请添加序号（如 `6.png`, `6.1.png`, `6.2.png`）。放映时按 **下方向键 (↓)** 即可呼出并无缝轮换，无图的页面按了也绝对静默。
6. **完成**：等待 AI 跑完，双击打开 `index.html`，享受你的汇报！

## ✨ 核心特性

- **极致单文件架构**：HTML, CSS 和原生 JS 全部封装在 `index.html` 内。完美适配大模型（LLM）的上下文窗口限制，AI 修改起来零负担。
- **cqw 绝对锁定排版**：全面采用最新的 CSS `cqw` 容器查询单位。强制锁定在 4:3 的优雅画幅下，无论你在 4K 带鱼屏还是 13 寸轻薄本上打开，文字和卡片的视觉比例都**绝对一致**，再也不会出现文字溢出。
- **原生高清 PDF 导出**：告别模糊的截图导出！点击页面右上角的 **[PDF]** 按钮，系统会调起浏览器原生打印面板。只需选择 `Microsoft Print to PDF` + `12x16in` 纸张 + `无边距`，瞬间获得可选中文字、无限放大的超清矢量 PDF 讲义。
- **智能图片弹窗与多图轮换**：内置全局图片弹窗系统，支持单图及多图连播。按下方向键 (↓) 一键优雅呼出当前页关联的高清原图。极客风的防尴尬静默机制，确保无图页面零报错、零打扰。
- **内嵌 AI 设计约束系统**：附带专属的 `web-presentation-design.md` 设计规范。这是写给 AI 看的“护城河”，严格规约了圆角、阴影、微动画、色彩（星巴克绿高级调色板）和留白，确保 AI 无论怎么自由发挥，都能输出具备高级商业质感的排版。

## 🛠️ 环境要求

- **浏览器**：最新版的 Chrome 或 Edge 浏览器（要求支持 CSS `@container`）。
- **工具**：任意一款得心应手的 AI 编程助手！
